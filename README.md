# ENTREGABLE-2-ED
Para compilar, primero hay que estar en el directorio donde se encuentren los archivos, y posteriormente utilizar el comando:

g++ main.cpp procesar_libros.cpp procesar_libros.hpp tinyxml2.cpp tinyxml2.h tree.cpp tree.hpp -o exe

En caso de usar main de pruebas propio, utilizar ese en lugar de main.cpp

Notar que en el main proporcionado el valor r de borrar_ratings se escribe como argumento en el llamado a la funcion.
